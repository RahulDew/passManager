# Abstractor The Password Manager
### Visit Demo - https://stasherdrive.netlify.app/

## Securely manage diverse account credentials, personal card information, and notes while conducting comprehensive health assessments of my credentials for optimal security.

### Tech Used :
#### Django -  A high-level Python web framework that enables rapid development of secure and maintainable websites.
#### SQLlite - Used to create a database, define tables, insert and change rows, run queries and manage an SQLite database file.
#### haveibeenpwned API V3 - The API allows the list of pwned accounts (email addresses and usernames) to be quickly searched via a RESTful service.
